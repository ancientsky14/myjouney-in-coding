Introduction Variables Functions Loops Arrays
Introduction

JavaScript is a powerful scripting language used to create dynamic websites and applications.

It runs in the browser and on the server using environments like Node.js.

console.log("Hello, world!");
  • Client-side scripting
  • Dynamic content
  • Interacts with HTML/CSS
Variables

Variables store data that can be used and manipulated later in the program.

JavaScript supports var, let, and const.

let name = "Alice"; const age = 30;
  • let: block-scoped
  • const: block-scoped, cannot be reassigned
  • var: function-scoped
Functions

Functions are reusable blocks of code that perform a specific task.

You can define functions using function declarations or arrow syntax.

function greet() { return "Hello"; }
  • Reusable code
  • Accepts parameters
  • Can return values
Loops

Loops allow you to repeat code multiple times based on a condition.

Common loops include for, while, and do...while.

for (let i = 0; i < 5; i++) { console.log(i); }
  • for loop
  • while loop
  • do...while loop
Arrays

Arrays are used to store multiple values in a single variable.

You can access array elements using their index.

let colors = ["red", "green", "blue"];
  • Zero-based indexing
  • Array methods like push(), pop(), shift()
  • Loop through arrays with forEach()
